DI in a nutshell

val kodein = Kodein {
  bind() with provider { RandomDice(0, 5) }
  bind() with singleton { SqliteDS.open("path/to/file") }
}

Controller(private kodein: Kodein) {
  val ds: DataSource = kodein.instance
}

Benefits

  • Reduced Dependencies

    Dependency injection makes it possible to eliminate, or at least reduce, a components unnecessary dependencies

  • Reduced Dependency carrying

    Prevents anti-patterns like passing instances through method calls

  • More Reusable Code

    Reducing a components dependencies typically makes it easier to reuse in a different context

  • 81% More Testable Code

    Easily replace dependencies with own mocks

  • More Readable Code

    Complex creation of dependencies is obsolete

Hype

Dependency Injection is everywhere! Frameworks are now available on almost all platforms and programming languages.
